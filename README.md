## Frontend

Angular 18 frontend for **Mettec Task Manager** using Bootstrap 5.

### Prerequisites

- **Node.js**: version 20.x or higher  
- **npm**: comes with Node.js  
- **Angular CLI**: version 18.x or higher

Check versions:

```bash
node -v
npm -v
ng version
```

### Setup

Run the following commands one by one:

```bash
cd frontend
npm install
npm build
npm start
```
### Run Unit Tests

```bash
npm test
```
